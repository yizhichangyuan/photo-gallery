# Photo Gallery

A beautiful photography portfolio website with waterfall layout, auto-scrolling columns, and search functionality.

## Features

- **Waterfall Layout**: Multiple vertical columns with auto-scrolling photos
- **Hover Interaction**: Hover to pause column and see photo title
- **Search**: Search photos by keywords
- **Download**: Download high-resolution photos
- **Responsive**: Adapts to different screen sizes (2-6 columns)
- **Smooth Animations**: 60fps GPU-accelerated scrolling

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is ready to deploy on Vercel:

1. Push to GitHub
2. Import to Vercel
3. Deploy

No additional configuration needed - Vercel will automatically detect Vite and build correctly.
